# Frontend–Backend Integration Spike

Purpose:
- Validate end-to-end flow from UI → Express → Prisma → PostgreSQL

What was proven:
- Frontend can call backend via env-based API client
- Express routes/controllers receive requests correctly
- Prisma successfully writes to Postgres (Docker)

Non-goals:
- Production-ready auth
- Security (hashing, tokens)
- Error handling polish

Status:
- Spike complete
- No further development planned on this branch
